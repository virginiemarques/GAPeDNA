# Changelog

All notable changes to this project will be documented in this file.

## [1.2.0] - 2026-03-05

### Changed

- Reference database updated to Midori GenBank 269 (2025-12-09) generated via
  CRABS v1.14 in silico PCR (replaces ecoPCR/ENA pipeline)
- Elasmobranches are now added, spatially and with the main primers used 
  for their detection
- IUCN Red List data refreshed (2026-03-05) using rredlist API v4; family
  classification now uses COL/GBIF backbone taxonomy
- Number of mismatches in the primers region is no longer displayed as the functionality 
  is not yet implemented in CRABS

## [1.1.2] - 2024-03-13

- Update to reference database ENA June 2023 (downloaded November 2023)
